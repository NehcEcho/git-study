## 学习建议

1. **动手实践**：每节课都要在电脑上实际操作
2. **理解原理**：不仅要知道命令，还要理解 Git 的工作原理
3. **多做练习**：完成每节课的课后练习
4. **日常使用**：在日常开发中使用 Git，熟能生巧

## 快速参考

### 最常用命令

```bash
# 初始化仓库
git init

# 添加文件
git add .

# 提交更改
git commit -m "提交信息"

# 查看状态
git status

# 查看历史
git log --oneline

# 创建分支
git checkout -b feature-branch

# 切换分支
git checkout main

# 合并分支
git merge feature-branch

# 推送到远程
git push origin main

# 拉取更新
git pull origin main
```

## 常见问题

### Q: 安装 Git 后命令找不到？
A: 需要重启终端，或将 Git 添加到系统 PATH

### Q: 提交时报错 "Please tell me who you are"？
A: 需要配置用户名和邮箱：
```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### Q: 如何退出 git log 的查看界面？
A: 按 `q` 键退出

### Q: 推送时报错 "failed to push some refs"？
A: 远程有更新，需要先 `git pull` 再推送
